TouchListeners

  • Touchstart keyrir einu sinni, gagnlegt fyrir setup
  • Touchmove er keyrt oft, á meðan við erum að færa til fingur (mús). Gott fyrir útreikninga.
  • Touchend keyrir einu sinni þegar fingur fer af eða útfyrir skjá (ath. ekki element). Gott fyrir cleanup og reset.
  • Touchcancel keyrir þegar touch event fer óvænt úrskeiðis (ytri ástæður t.d. hardware event (takki), screen orientation, of margir fingur), einskonar try/catch virkni.

Gerðu eftirfarandi:

  1. Stilltu á Mobile view í dev mode (F12) í vafra og hafðu console opið.
  2. smelltu á músina og haltu takkanum niðri án þess að hreyfa. touchstart
  3. færðu svo músina til á skjánum. touchmove
  4. slepptu músatakkanum. touchend
  5. skoðaðu afar lauslega í console TouchEvent objectið. t.d target, timeStamp og Touchlists (touches, changedTouches, targetTouches)
